Final Project Proposal

Project Overview:

For my final project, I will attempt to develop a real-time renderer in C++, utilizing the OpenGL library. It will be built from the ground up, requiring me to implement everything from the basic canvas to render to, to the entire process of generating, shading, and transforming models. A more exhaustive list of features are listed below.

  1. Canvas creation
  2. User input (keyboard, mouse, scroll wheel)
  3. Coordinate Systems (local, world, camera, clip, screen)
  4. Controllable Camera
  5. Model importations and Model/Mesh creation
  6. GLSL shaders (vertex, geometry [exploding/reconstructing, fur (visualizing normals)], and fragment)
  7. Textures
  8. Basic Lighting
  9. Environment maps (mirror, phong)
  10. Shadow mapping (percentage-closer filtering for antialiasing for soft shadows)
  11. Ambient Occlusion (SSAO)

1-7 are complete at the time of the proposal submission (with the exception of geometry shaders). 8-11 are not yet completed.


Project Requirements:

This project aligns with project 3.2 “Basic High Quality Rendering.” Note that all of the project requirements are met:

  1. GLSL Shaders are implemented (the non-standard effects are implemented via geometry shaders).
  2. Shadow mapping is implemented
  3. Environment mapping is implemented
  4. The two add-ons are (1) Phong shading with environment maps and (2) Ambient Occlusion (SSAO)

Next Steps:

Over the next week, I will continue implementing each of the aforementioned features, starting first with basic lighting, then environment maps, then shadow mapping, then ambient occlusion and lastly geometry shaders. I hope to render each of these components in a single, real-time program, but will also develop individual programs to demonstrate each major feature separately.
