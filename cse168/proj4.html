Importance Sampling, Assignment 4:

This project is based off of the previous Ray Tracing, Assignment 1 project, Direct Lighting, Assignment 2 project, and Indirect Path Tracer, Assignment 3 project. Just as in Assignment 1, 2 and 3, the codebase was written in C++ with no external graphics libraries (e.g. Optix, OpenGL). All of the library functionality for vectors, matricies, and their corresponding mathematical operations were implemented by me. To speed up testing, the openmp multithreading tool was utilized.

The goal of this assignment was to implement importance sampling to manipulate the pdf to produce good results for any material. Multiple importance sampling, which allows multiple sampling methods to be used based on which techniques work best in different situations is also implemented.

I found this assignment to be extremely difficult to debug, and I was disappointed that I was unable to pass all of the test cases. Nevertheless, I implemented all of the sampling methods: cosine sampling, brdf importance sampling, GGX Microfacet important sampling, and multiple importance sampling.
