Final Project

Project Overview:

For my final project, I developed a real-time renderer in C++, utilizing the OpenGL library. It was built from the ground up, requiring me to implement everything from the basic canvas to render to, to the entire process of generating, shading, and transforming models. A more exhaustive list of features are listed below.

  1. Canvas creation
  2. User input (keyboard, mouse, scroll wheel)
  3. Coordinate Systems (local, world, camera, clip, screen)
  4. Controllable Camera
  5. Model importations and Model/Mesh creation
  6. GLSL shaders (vertex and fragment)
  7. Phong Shading
  8. Textures
  9. Basic Lighting (Directional and Point)
  10. Environment maps (mirror reflections, diffuse through uniform hemispherical sampling)
  11. Shadow mapping
  12. Diffuse and Specular Mapping (through textures)

Project Requirements:

This project aligns with project 3.2 “Basic High Quality Rendering.” Note that all of the project requirements are met:

  1. GLSL Shaders are implemented (difficult effects without glsl shaders include phong shading, diffuse and specular mapping, and diffuse environment mapping).
  2. Shadow mapping is implemented
  3. Environment mapping (reflective) is implemented
  4. The two add-ons are (1) Diffuse lighting with environment maps and (2) diffuse and specular mapping

Final words

Hopefully this summer I can implement the rest of the features (and more). Thank you to the entire CSE 168 faculty.
