Direct Lighting, Assignment 2:

This project is based off of the previous Ray Tracing, Assignment 1 project. Just as in Assignment 1, the codebase was written in C++ with no external graphics libraries (e.g. Optix, OpenGL). All of the library functionality for vectors, matricies, and their corresponding mathematical operations were implemented by me.

The goal of this assignment was to implement direct lighting of area light sources for the ray tracer with accurate phycially-based rendering. The core theory behind realistic direct lighting was the rendering equation. By using the rendering equation, two techniques were employeed: analytic and Monte Carlo sampling. The analytic solution was noise-free, but could only render simple scenes with no shadows. The Monte Carlo solution allowed for soft shadows because it sampled multiple points per pixel rendered on the area light. To reduce noise, stratified sampling, was employeed.

As for my personal impression of this assignment, I found it to be very satisfying. This was the first time I implemented physically-based lighting, and was a huge milestone in terms of my computer graphics journey. While the theory was very heavy in mathematics (both requiring Calculus and Linear Algebra), it was nice seeing all of the math I learned through school being used to develop something real and tangible.
