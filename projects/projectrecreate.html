Project Overview

In order to practice creating larger-scale Java projects that require design patterns, event-driven programming, and UI design, I started recreating some of my favorite video games. I started with Tic Tac Toe, then I learned how to make platformer video games and attempted to create Super Mario Bros. While my version of Super Mario Bros is still in it's developmental stage, I had a lot of fun working on this project and learned a ton from it.
